# 📱 Projeto

## Título
`Receitas Cotidianas`

---

## Objetivo do Aplicativo
O objetivo do aplicativo criado é facilitar o acesso a receitas de forma simples e prática, reunindo quatro receitas diferentes para que o usuário possa explorar e escolher o que deseja preparar. Além de apenas visualizar o passo a passo, o aplicativo também permite que as pessoas deixem comentários em cada receita, o que torna a experiência mais interativa, já que os usuários podem compartilhar opiniões, dicas ou até sugestões de melhorias. Dessa forma, o app não serve apenas como um guia de receitas, mas também como um espaço de troca de experiências entre os usuários, tornando o processo de aprender a cozinhar algo mais dinâmico, útil e interessante.

## Funcionalidade do Aplicativo
O aplicativo possui diversas funcionalidades que tornam seu uso simples e intuitivo para o usuário. Entre elas, é possível rolar a tela para baixo (scroll) para visualizar o restante das receitas disponíveis, facilitando a navegação dentro do app. Além disso, há a mudança de telas entre as diferentes receitas, permitindo que o usuário acesse cada uma de forma organizada e separada. O aplicativo também conta com um sistema de comentários, onde os usuários podem compartilhar suas opiniões, dicas e experiências sobre cada receita. Por fim, existe um sistema de retorno para a tela inicial, que ajuda o usuário a voltar facilmente ao começo do aplicativo e continuar explorando outras opções disponíveis.
