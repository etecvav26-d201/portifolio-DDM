# 📱 Portifólio Desenvolvimento de Dispositivos Móveis - D201

Bem-vindo ao repositório **Portifólio Desenvolvimento de Dispositivos Móveis**! 🚀  

Este repositório reúne diversos **projetos e aplicações mobile**, desenvolvidos com o 
objetivo de **aprender, praticar e demonstrar conhecimentos na criação de aplicativos para celular**.

Aqui você encontrará **diferentes tipos de aplicativos**, criados utilizando o **MIT App Inventor**, 
uma plataforma intuitiva que permite o desenvolvimento de apps por meio de **programação em blocos**.  
Os projetos presentes neste portfólio representam a evolução do aprendizado ao longo do estudo de 
**desenvolvimento mobile, lógica de programação e design de interfaces**.

O desenvolvimento dos aplicativos foi baseado tanto nos **conceitos apresentados na apostila** quanto 
nos **conteúdos trabalhados em aula**, garantindo uma aprendizagem prática e aplicada.

O objetivo deste repositório é:

- 📚 Servir como **registro de aprendizado em desenvolvimento mobile**
- 📱 Reunir **aplicativos desenvolvidos no MIT App Inventor**
- 🧠 Demonstrar **lógica de programação aplicada em dispositivos móveis**
- 🎨 Explorar **criação de interfaces interativas**
- 🛠️ Praticar **desenvolvimento de funcionalidades para apps**
- 📂 Manter um **portfólio organizado de projetos mobile**

# 🎯 Objetivo do Projeto

O objetivo principal deste portfólio é **consolidar o aprendizado em desenvolvimento de aplicativos mobile**, reunindo 
projetos que demonstram:

- Criação de aplicativos funcionais
- Aplicação da lógica de programação
- Interação com o usuário
- Uso de recursos do dispositivo (toque, sensores, mídia)
- Desenvolvimento visual de interfaces
- Organização e estruturação de projetos

Além disso, o repositório funciona como uma **base de referência para estudos futuros e evolução no desenvolvimento mobile**.
